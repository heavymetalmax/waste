# PL V2 Project

This repository is organized exclusively for the **pl_v2** project - a Polish language website for biogas and waste management solutions.

## Project Structure

```
v2/                 # Main project directory
├── index.html       # Homepage
├── case-lubin.html  # Case study
├── chat.html        # Chat page
├── mission.html     # Mission page
├── technology.html  # Technology page
├── whitepaper.html  # Whitepaper
├── teo.html         # TEO page
├── kalkulator-roi.html  # ROI Calculator
├── design-guide.html    # Design guide
├── design-system.css    # Design system styles
├── DESIGN_SYSTEM.md     # Design system documentation
└── partials/        # Reusable components

design-system/      # Shared design system assets
img/                # Images and media assets
translations/       # Translation files

_archive/           # Archived projects and legacy files
├── other-projects/  # Other unmaintained projects
├── scripts/         # Python automation scripts
├── old-archive/     # Previous archive contents
└── todo/            # Legacy task lists
```

## Working with v2

All active development should be done in the `v2/` directory. Shared assets (images, design system) are available in the root-level `design-system/` and `img/` directories.

### Key Files
- **v2/design-system.css** - Main styling
- **design-system/design-guide.html** - Visual design reference
- **v2/DESIGN_SYSTEM.md** - Design documentation

## Archive

All other projects, legacy code, and automation scripts have been moved to `_archive/` to keep the workspace focused on v2 development.
