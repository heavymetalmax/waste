## Hub & Spoke — повний розрахунок

**Структура**: 1 hub (30K РЕ) + 4 spokes (5K РЕ)

| Параметр | Значення |
|----------|----------|
| Осад hub | 985 т/рік |
| Осад 4 спайків | 730 т/рік |
| РАЗОМ HTC | 1,715 т/рік |

**Де hub заробляє:**

```
А) Власна економія:
   Стара вартість:   PLN 542,025
   HTC OPEX:        -PLN 208,433
   Чиста економія:  +PLN 333,592  ← ГОЛОВНИЙ СТИМУЛ для hub

Б) Gate fee від спайків: +PLN 146,000
   Processing spokes:   -PLN 154,395
   Чисто від спайків:   -PLN 8,395

РАЗОМ net benefit hub: PLN 325,197/рік = EUR 81,299/рік
```

**CapEx**: PLN 15M; грант 85% = PLN 12.75M; внесок hub = PLN 2.25M (EUR 562K)
**Payback для hub**: **6.9 років** при 85% гранті

## Вигода для малої гміни (spoke, 5K РЕ) — СКОРИГОВАНИЙ РОЗРАХУНОК

**Важливо**: транспорт НЕ включено в gate fee — spoke сплачує окремо.

```
Поточні витрати:  PLN 100,375/рік (PLN 550/т × 182 т)

Нові витрати:
  Gate fee BTC:       PLN 200/т × 182 т  = PLN 36,400/рік
  Транспорт (50 км):  PLN 150/т × 182 т  = PLN 27,300/рік
  РАЗОМ:                                   PLN 63,700/рік
─────────────────────────────────────────────────────
Щорічна економія:    PLN 36,675/рік = EUR 9,169/рік (36%)
За 10 років:         PLN 366,750    = EUR 91,688

Транспорт 20 км:  PLN 60/т → PLN 23,700/рік → економія PLN 40,975/рік (41%)
Транспорт 100 км: PLN 300/т → PLN 91,600/рік → економія лише PLN 8,775/рік (9%)
```

**Висновок**: Hub повинен знаходитись **максимум 50 км** від спайків.
При 50+ км — пропозиція втрачає привабливість. При 100 км — майже немає економії.

**Pitch для spoke (скоригований)**:
> "Підпишіть 10-річний договір. Ваш повний нова вартість (gate fee + транспорт 30 км) = PLN 300/т vs PLN 550/т зараз = **45% економія**."
