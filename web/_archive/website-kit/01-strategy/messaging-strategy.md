# Messaging Strategy — BTC Consulting Website

## Позиціювання

BTC не продає compliance. BTC продає **вікно можливостей**, яке закривається.

Регуляторний тиск — це тригер уваги, але не головний аргумент. Сам по собі він викликає параліч ("надто дорого, потім"). Правильна комбінація: страх + виход + дедлайн на виход.

---

## Ієрархія меседжів

### 1. HOOK — Регуляторний тригер
> "До 2027 кожна oczyszczalnia повинна мати рішення для осаду."

Функція: захопити увагу. Використовується в Hero кожної сторінки.
Аудиторія: WWTP-оператори, керівники гмін.

### 2. REFRAME — Гранти як вікно
> "HTC-установка коштує X. З грантами NFOŚiGW/LIFE — 0.4X. Але гранти доступні зараз, не через 3 роки."

Функція: перетворити "надто дорого" на "надто дешево, щоб пропустити".
Деталі:
- PLN 30B грантів доступно (NFOŚiGW + EU LIFE + Horizon)
- Покриття: 40-60% capex
- First-come-first-served: ранні проекти отримують гранти, пізні — платять повну ціну
- Грантова заявка потребує 6-12 місяців підготовки → треба стартувати зараз

### 3. EXIT — BTC як єдиний реалістичний шлях
> "Ми єдина польська платформа з власною HTC-технологією і 10+ роками досліджень разом з AGH."

Функція: закрити питання "чому BTC, а не хтось інший".
Деталі:
- TerraNova = 1 проект в Польщі, іноземна
- Veolia/SUEZ = без HTC
- BTC = власне IP (HTC-S, HTC-D, TH+HTC) + AGH + офіційний статус HTC з 2025

### 4. SCARCITY — Обмежена пропускна здатність
> "BTC бере 8-10 нових проектів на рік. Якщо зачекати — черга."

Функція: urgency без регуляторного тиску. Підштовхує до дії тут і зараз.
Важливо: це має бути правдою — не маркетинговий трюк.

## Додаткові кути (за аудиторією)

### Для малих гмін (<10K PE)
**Hub & Spoke як демократизатор**
> "Ваша гміна не може дозволити власний реактор. Але в моделі 3-5 гмін на один хаб — може."

Функція: відкриває ринок, який без цього аргументу не конвертує.

### Для інвесторів
**Осад як commodity**
> "Ваш осад зараз коштує вам гроші. BTC перетворює його на ресурс з ринковою вартістю."

Функція: ре-фреймінг з compliance play на commodity/asset play.
Підтекст: хто контролює потік осаду — контролює ринок.

### Для всіх
**Польська технологія**
> "Не імпортоване рішення. Власна польська розробка, адаптована під польське законодавство і польські WWTP."

Функція: патріотизм + локальна підтримка + регуляторна точність.

## Чого уникати

- Не починати з технічних характеристик HTC (втрачає неінженерну аудиторію)
- Не акцентувати "штрафи та каки" без виходу (параліч рішення)
- Не позиціювати як "консалтингову компанію" — [[strategy-reframe]]
- Не обіцяти 80% грантовий success rate без підтвердження — [[evidence-gaps]]
