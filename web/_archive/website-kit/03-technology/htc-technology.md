## Три власні продукти

### 1. HTC-S — для "пост-біологічного" осаду

**Цільовий сегмент**: WWTPs без метанового зброджування (більшість малих польських ОС)

**Процес**: 200-210°C, ~2.5 MPa, до 2 годин → структурна трансформація органіки

**Виходи**:
- **HTC liquid** → направляється до власного або існуючого метанового зброджування
- **Hydrochar** (30-40% вологи) → кількість в 4+ рази менша від вихідного осаду
- **HTC gas** → після очищення у water scrubber → в когенераційний агрегат або котел

**Енергетика**: Вироблений біогаз достатній для генерації тепла для власних потреб установки.

---

### 2. HTC-D — для стабілізованого осаду

**Цільовий сегмент**: WWTPs з існуючим аеробним або анаеробним (метановим) зброджуванням

**Процес**: Ідентичний HTC-S, але вхід — вже стабілізований осад

**Виходи**: Ідентичні HTC-S

---

### 3. TH+HTC — для ОС з метановим зброджуванням

**Цільовий сегмент**: Великі WWTPs (>15K RLM) з існуючим AD

**Процес**: Двоетапний —
1. **TH (Thermal Hydrolysis)**: 160-170°C, 0.8-1.0 MPa, до 1 год → гідролізат + різкая декомпресія
2. **HTC**: після метанового зброджування

**Виходи**:
- **Hydrolyzate** → 35-50% більше біометану від зброджування
- **HTC liquid** → додатково до метанового зброджування або окремо
- **Hydrochar** → ~5x менший від вихідного осаду
- **TH+HTC gas** → когенерація або котел

**Енергетика**: ⭐ TH+HTC установка виробляє достатньо для генерації ЕЛЕКТРОЕНЕРГІЇ та тепла для власних потреб. Повна енергетична самодостатність!

## Регуляторна основа

**З 14 січня 2025** — анаеробний HTC-процес офіційно дозволений для стабілізації осаду в Польщі.
Умова: необхідно знизити вміст органічних речовин в сухій речовині осаду мінімум на **38%**. HTC знижує значно більше.

Це означає: всі польські ОС, які зараз використовують застарілі методи стабілізації, можуть законно переходити на HTC з 01/2025.

## Конкурентне значення

1. **BTC — єдина польська компанія з власними HTC-продуктами для осаду**: TerraNova (Німеччина) продає реактори; Cambi (Норвегія) продає TH-системи; ніхто не має інтегрованого польського рішення з 10-річною AGH-підтримкою.

2. **Три версії = три ринкових сегменти**: HTC-S для малих ОС без AD; HTC-D для середніх з AD; TH+HTC для великих = BTC може обслуговувати весь ринок від 1K до 150K RLM.

3. **TH+HTC — унікальний продукт**: повна енергетична самодостатність + 35-50% більше біогазу. TerraNova пропонує TH окремо від HTC; BTC інтегрує їх матеріально та енергетично.
