# Фаза 2: Трансформація сайту в інженерний портал (SupremSEO)

Цей документ містить покроковий план переходу сайту до Фази 2 (перетворення на B2B-портал для генерації лідів) та інструкції щодо використання SEO-інструментів SupremSEO для мінімізації витрат.

---

## ЧАСТИНА 1. Покроковий план розгортання «Фази 2»

### Етап 1: Інфраструктура та Перенесення (Тиждень 1–2)
* **Домен та хостинг:** Купівля власного домену (наприклад, `biotc.com.ua` чи `biotc-engineering.com`). Перенесення коду з GitHub Pages на стабільний хостинг.
* **Мультимовність:** Налаштування правильної структури папок для мов: `/` (UA), `/en/` (EN), `/pl/` (PL).
* **Корпоративна пошта:** Налаштування пошти виду `office@biotc.com.ua` замість безкоштовних скриньок.

### Етап 2: Розширення структури (Тиждень 3–5)
Перехід від однієї сторінки до архітектури, яка здатна ранжуватися в Google. Створення нових сторінок:
* **Каталог обладнання:** Окремі сторінки під кожну модель установки (наприклад, *BioTC HTC-S5*, *HTC-S15*).
* **Рішення для галузей:** Спеціалізовані сторінки: *«Для міських водоканалів»*, *«Для харчової промисловості»*, *«Для агрокомплексу»*.
* **Сторінка кейсу (Любін):** Детальний розбір реалізованого проєкту з цифрами, фото, технічними викликами та їх вирішенням.

### Етап 3: Наповнення, ТЕО та Конверсія (Тиждень 6–8)
* **Професійний контент:** Написання глибоких технічних текстів для кожної сторінки.
* **Lead Magnet (Магніт для лідів):** Розробка «Опитувального листа для розрахунку ТЕО» у форматі PDF, який замовник може завантажити, залишивши контакти.

## ЧАСТИНА 2. Як використовувати SupremSEO для реалізації Фази 2

Для виконання цього плану вам знадобляться всього 3-4 інструменти з пулу SupremSEO. Працюйте з ними «пакетами»: зайшли, зібрали дані, вивантажили в Excel, вийшли.

### Крок 1: Збір «B2B/B2G» семантики (Ключових слів)
> **Інструмент:** **Semrush** або **Ahrefs** (розділ *Keyword Wizard* / *Keyword Explorer*).

Вам потрібні не просто масові запити, а дорогі, вузькоспеціалізовані ключові слова.
1. Введіть базові терміни: *«утилізація мулу»*, *«очисні споруди обладнання»*, *«гідротермальна карбонізація»*, *«осад стічних вод»*, *«обезводнення осаду»*.
2. Зберіть усі комерційні та технічні хвости (наприклад: *«модернізація очисних споруд водоканалу»*).
3. Отримайте дані про частотність в Україні, Польщі та ЄС.

### Крок 2: Шпигунство за європейськими конкурентами
> **Інструмент:** **Ahrefs** або **Semrush** (розділ *Site Audit* / *Competitive Research*).

1. Знайдіть через Google європейські компанії (особливо німецькі/польські), які займаються HTC або утилізацією біовідходів.
2. Вставте їхні URL в інструмент SupremSEO.
3. Проаналізуйте:
   * Їхні трафікові ключові слова.
   * Структуру сайту.
   * Схеми та графіки.
4. Скопіюйте логіку архітектури для Фази 2.

### Крок 3: Написання ТЗ для копірайтера або ШІ-генерація
> **Інструмент:** **Surfer SEO** або **Jasper**.

Тексти мають бути експертними, з високим рівнем E-E-A-T.
1. Введіть головне ключове слово в Surfer SEO.
2. Отримайте інструкцію: обсяг тексту, `H2-H4` заголовки, технічні терміни.
3. Напишіть текст з урахуванням цих вимог.

### Крок 4: Технічний аудит перед релізом
> **Інструмент:** **Semrush Site Audit** або **Screaming Frog SEO Spider**.

Перевірте багатосторінковий сайт перед показом клієнтам та індексацією:
1. Запустіть повний технічний аудит.
2. Перевірте: биті посилання, відсутні зображення, мета-теги для 3 мов (UA/EN/PL), дублі сторінок.
3. Виправте помилки перед релізом.

### Чек-лист «Розумного використання»
- [ ] **Ніяких конфіденційних даних:** Не вводьте назви клієнтів (українських/польських) або закритих тендерів. Шукайте лише загальні терміни.
- [ ] **Експорт у хмару:** Одразу вивантажуйте дані в Excel/Google Таблиці, оскільки групові акаунти можуть переривати роботу.
