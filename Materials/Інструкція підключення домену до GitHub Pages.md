# Інструкція: підключення кастомного домену до GitHub Pages

## Що у вас є
- Репозиторій: `https://github.com/heavymetalmax/waste`
- Сайт зараз доступний на: `https://heavymetalmax.github.io`
- Потрібно: підключити `biotc.pl` (або інший домен)

---

## Крок 1 — Купити домен

Рекомендовані реєстратори:

| Реєстратор | Домени .pl | Посилання |
|---|---|---|
| OVH | ~40 PLN/рік | ovh.pl |
| Namecheap | ~15 $/рік | namecheap.com |
| Domeny.pl | ~30-50 PLN/рік | domeny.pl |

При реєстрації вкажіть будь-який email — він потрібен для управління доменом.

---

## Крок 2 — Налаштувати DNS у реєстратора

Після купівлі домену зайдіть в панель управління DNS вашого реєстратора та додайте **4 A-записи** і **1 CNAME-запис**:

### A-записи (для apex домену `biotc.pl`)

| Тип | Ім'я | Значення |
|---|---|---|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |

### CNAME-запис (для `www.biotc.pl`)

| Тип | Ім'я | Значення |
|---|---|---|
| CNAME | www | heavymetalmax.github.io |

> **Примітка:** зміни DNS поширюються від 30 хвилин до 48 годин.

---

## Крок 3 — Налаштувати GitHub Pages

1. Відкрийте: `https://github.com/heavymetalmax/waste/settings/pages`
2. У розділі **"Custom domain"** введіть: `biotc.pl`
3. Натисніть **Save**
4. Поставте галочку **"Enforce HTTPS"** (з'явиться через ~10 хвилин після збереження)

GitHub автоматично:
- Створить файл `CNAME` в репозиторії
- Видасть безкоштовний SSL сертифікат (Let's Encrypt)

---

## Крок 4 — Оновити посилання на сайті

Після підключення домену потрібно оновити кілька місць у коді:

### 4.1 Файл `build.py` — hreflang посилання
У шаблонах `biotc/index.html` є рядки:
```html
<link rel="alternate" hreflang="uk-UA" href="https://biotc.com.ua/" />
```
Замінити `biotc.com.ua` на ваш реальний домен.

### 4.2 Додати `sitemap.xml`
Після підключення домену — запросіть у Claude Code і він згенерує автоматично.

### 4.3 Google Search Console
Зареєструйте домен на `search.google.com/search-console` — дозволить Google проіндексувати сайт.

---

## Перевірка

Через 1-24 год після виконання всіх кроків:

1. Відкрийте `https://biotc.pl` — має показати ваш сайт
2. Відкрийте `https://www.biotc.pl` — має редиректити на `biotc.pl`
3. Поруч з адресою має бути замок 🔒 (HTTPS)

---

## Якщо щось не так

**Сайт не відкривається через 24 год:**
- Перевірте A-записи в DNS (саме 4 IP адреси GitHub)
- Перевірте що в GitHub Settings → Pages вказаний ваш домен

**HTTPS не працює:**
- В GitHub Settings → Pages натисніть "Enforce HTTPS" знову
- Зачекайте ще 10 хвилин

**Питання:** зверніться до Claude Code з повідомленням про помилку.

---
